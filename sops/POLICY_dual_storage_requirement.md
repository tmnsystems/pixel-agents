# Dual Storage Policy for AuntTBot

## Policy Established: February 12, 2026

**REQUIREMENT**: All important information must be stored in BOTH locations:
1. **Internal AI memory** (for conversation continuity)
2. **Tele_ files** (for permanent access and local agent processing)

## What Must Be Dual-Stored:
- Plans and SOPs
- Goals and facts about Tina
- Technical discoveries from transcript analysis
- System integration opportunities
- Business insights and strategies
- Workflow improvements
- Security considerations

## File Naming Convention:
- `Tele_MEMORY_[topic].md` - For general memory storage
- `Tele_SOP_[process].md` - For standard operating procedures  
- `Tele_TODO_[action].md` - For action items and tasks
- `Tele_ANALYSIS_[source].md` - For transcript/content analysis
- `Tele_POLICY_[rule].md` - For policies and requirements

## Protection Against Data Loss:
- Never rely solely on internal memory
- Critical information survives disconnections
- Local agent can access all stored knowledge
- Searchable file system for historical reference

**This policy is now in effect for all future interactions.**
